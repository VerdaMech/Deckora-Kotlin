# Deckora

Deckora es una aplicación móvil que permite a los usuarios gestionar y organizar cartas TCG dentro de carpetas personalizadas.

## Flujo de la aplicación

1. El usuario ingresa a la pantalla de perfil.  
2. Inicia sesión o crea una cuenta.  
3. Accede a la sección Carpetas y crea una carpeta.  
4. Registra una carta TCG subiendo una imagen desde la galería o la cámara.  
5. Guarda la carta dentro de una carpeta.  
6. Puede revisar:
   - Sus cartas dentro de las carpetas.
   - Las cartas más destacadas en la pantalla Home.
