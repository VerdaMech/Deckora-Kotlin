Funcionalidades del proyecto:
  -Crear cartas desde la funcionalidad de tomar una foto
  -Crear cartas desde la funcionalidad de abrir la galeria
  -Crear carpetas estilo galería iphone para organización
  -Registro de usuarios
  -Login de usuarios
  -Listar cartas Top del momento
  
Endpoints utlizados:
  -Utilizamos el framework de java Spring, para realizar nuestra Api y conexión a nuestra base de datos
  -Link a nuestros endpoints y metodos de la api: https://deckora-api-kotlin-z6za.onrender.com/doc/swagger-ui/index.html

Pasos para ejecutar:
  -Dentro de la aplicación dirigirse a la barra inferior y hacer click en "Tú Perfil".
  -Si es un nuevo usuario apretar el botón de "Crear usuario" y si ya esta registrado apretar el de "Iniciar Sesión".
  -Una vez iniciada sesión puedes empezar a cargar tus cartas en el apartado de "Cámara" en la barra inferior.
  -Dentro de la pantalla de Cámara tienes la opción de subirla a través de la galería o tomando una foto.
  -Pueder revisar tus cartas guardadas en el apartado de "Carpetas" en la barra inferior.
  -Dentro de la pantalla de Carpetas puedes crear nuevas con el botón "+" que se encuentra en la esquina inferior izquierda, para una organización más personalizada.
